Percobaan position: relative

Relative akan dipindah dari barisan normal flow ke atas, jadi menggunakan top bukan margin-top

Percobaan position: absolute

absolute seperti relative, namun elemen bawahnya akan menganggap elemen tsb tidak ada (menimpa tempat)

sebagai contoh pink akan dikasih margin-left 20px, dan ungu akan absolute

fourth (ungu) ditarik dari normal flow dan fifth (pink) mengambil posisi milik fourth (ungu)
